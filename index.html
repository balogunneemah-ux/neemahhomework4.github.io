Quantitative Methods Implemented

Poisson Count Models

Poisson regression models are used when the dependent variable represents a count of events. In political science, this includes protest frequency, campaign contact counts, or instances of political participation. 

poisson_model <- glm(protest_count ~ campaign_contacts,
                     family = poisson(link = "log"),
                     data = df)

Multilevel Models

Multilevel modelling accounts for hierarchical data structures, such as individuals nested within countries. This approach allows for estimation of both fixed and random effects. 

library(lme4)
model <- lmer(trust ~ 1 + (1 | country), data = df)

Factor Analysis

Factor analysis identifies latent dimensions underlying survey items. This technique reduces dimensionality and captures hidden constructs such as political trust or ideological orientation. 

fa <- factanal(~ item1 + item2 + item3 + item4,
               factors = 1,
               data = survey_data)
